
ENTERPRISELIBRE SYSTEM MANAGER
------------------------------


EnterpriseLibre integrates more than 50 of the best open source components into a full-stack enterprise cloud, including virtual desktops, that can replace almost all proprietary software for up to 1,000 users on a single server.  This image provides a System Manager cloud that can create other enterprise clouds, enabling anyone to offer complete cloud solutions to others.

Installation instructions can be found in INSTALL.html

FIRST CHECK IF THERE IS A NEWER VERSION and find other documentation and resources at: https://wiki.enterpriselibre.org/index.php/Development
